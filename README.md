Simple cli calculator based on reverse polish notation. Version: 0.1.0

<h1 align="center">Make</h1>

To run tests:

```
$ make check
```

To build:

```
$ make
```

To install:

```
$ sudo make install
```

To uninstall:

```
$ sudo make uninstall
```

To clean:

```
$ make clean
```

<h1 align="center">Usage</h1>

```
$ c "2+2*2"
> 6
```
```
$ c "2-"
> Error: Invalid syntax.
```
```
$ c "2/0"
> Error: Division by zero.
```
